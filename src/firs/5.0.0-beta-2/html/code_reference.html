Obtaining Source

Source checkout available from the FIRS Repo on GitHub.

FIRS is licensed under GPL(v2).


Compiling

FIRS source includes a makefile for convenient building. This requires 'Make', typically provided by the ports tree on Linux / Unix, Xcode on macOS, and other build systems on Windows.

Other dependencies include:

  • Python 3.11+
  • NML 0.7.5 or newer (NewGRF compiler)
  • Chameleon (Template engine)
  • Markdown (Markdown text format parser)
  • Graphviz (optional, used to generate cargo flow charts in html docs)
