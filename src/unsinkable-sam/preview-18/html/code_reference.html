CargoLiner Subclass

Piece goods cargos, other selected cargos. Equivalent of Road Hog box hauler and Iron Horse box car / van. IRL: "GCV", "Break-bulk", "Pallet carrier", also cargo packet (but not mail packet) Not "box ship" because IRL they are container carriers (yair). The name is ultimately just a generic solution to 'this is not quite a general freighter', that doesn't describe the specific shape or cargo type or RL trade name. Many name variants were tried, Cargo Liner seemed to stick.
